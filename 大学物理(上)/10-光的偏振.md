# 10. 光的偏振

## 10.1. 自然光与偏振光的基本概念

### 10.1.1. 光的偏振现象
- **10.1.1.1. 光矢量与偏振**
    - 光是电磁波，其电场强度矢量 $\vec{E}$ 对感光物质起主要作用，称为**光矢量**。
    - 光矢量的振动方向与光的传播方向垂直，这一特性称为**光的偏振**。
    - 偏振是横波特有的现象，纵波没有偏振问题。
- **10.1.1.2. 自然光 (Natural Light)**
    - 定义：普通光源发出的光，其光矢量在垂直于传播方向的平面内沿各个方向振动，且振幅均等，各方向振动无固定相位关系。
    - 表示：可分解为两个振动方向相互垂直、振幅相等、无固定相位差的独立光振动分量。平面表示法中用垂直于传播方向的点和短线组合表示。
- **10.1.1.3. 偏振光 (Polarized Light)**
    - **线偏振光 (Linearly Polarized Light / 完全偏振光)**
        - 定义：光矢量只沿某一固定方向振动。
        - 振动面：偏振光的振动方向与光的传播方向构成的平面。
        - 表示：若振动在纸面内用短线，垂直纸面用点。
    - **部分偏振光 (Partially Polarized Light)**
        - 定义：光矢量在某一方向的振动较其垂直方向的振动占优势。
        - 表示：某一方向的点或短线数量多于另一方向。
    - **椭圆偏振光与圆偏振光**
        - 定义：光矢量末端在垂直于传播方向的平面上描绘出椭圆（椭圆偏振光）或圆（圆偏振光），同时光波向前传播。
        - 旋向：根据光矢量旋转方向分为左旋和右旋。

## 10.2. 起偏、检偏与马吕斯定律

![](./assets/graph10-1.png)

### 10.2.1. 起偏与检偏
- **10.2.1.1. 定义**
    - **起偏 (Polarization)**：将自然光转变为偏振光的过程。所用元件称为**起偏器 (Polarizer)**。
    - **检偏 (Analysis)**：检验光是否为偏振光以及确定其偏振状态的过程。所用元件称为**检偏器 (Analyzer)**。
- **10.2.1.2. 偏振片 (Polaroid)**
    - 原理：利用某些物质（如电气石、硫酸奎宁晶体）对不同方向光振动的选择性吸收（二向色性）。
    - 偏振化方向：偏振片允许通过的光振动方向。
    - 功能：同一偏振片既可作起偏器，也可作检偏器。
- **10.2.1.3. 不同类型光的检验方法**
    - **自然光**：通过起偏器后光强减半 ($I = I_0/2$)，变为线偏振光。旋转起偏器，透射光强不变。
    - **线偏振光**：通过检偏器后，透射光强随检偏器旋转 ($I = I_0 \cos^2\alpha$)，出现两次最强和两次消光。
    - **部分偏振光**：通过检偏器后，透射光强随检偏器旋转，出现两次最大和两次最小，但不消光。

### 10.2.2. 马吕斯定律 (Malus's Law)
- **10.2.2.1. 定律内容**
    - 当一束强度为 $I_0$ 的线偏振光通过一个偏振片（检偏器）时，若入射光的振动方向与偏振片的偏振化方向成 $\alpha$ 角，则透射光的强度 $I$ 为：
      $I = I_0 \cos^2\alpha$
- **10.2.2.2. 应用实例**
![](./assets/graph10-2.png)
![](./assets/graph10-3.png)
    - **例1：三偏振片系统**
![](./assets/graph10-4.png)
        - **问题**：自然光通过两偏振化方向夹角为 $60^\circ$ 的偏振片P1、P2，透射光强为 $I_1$。在P1、P2间插入偏振片P3，P3与P1、P2的偏振化方向夹角均为 $30^\circ$。求此时透射光强 $I$。
        - **解**：
            - 无P3时：$I_1 = (I_0/2) \cos^2 60^\circ = I_0/8$。
            - 有P3时：光通过P1后强度为 $I_0/2$。通过P3后强度为 $(I_0/2)\cos^2 30^\circ$。再通过P2后强度为 $I = [(I_0/2)\cos^2 30^\circ] \cos^2 30^\circ = (I_0/2)(\frac{\sqrt{3}}{2})^4 = \frac{9}{32}I_0$。
            - $I = \frac{9/32}{1/8} I_1 = \frac{9}{4}I_1$。
    - **例2：混合光的成分分析**
        - **问题**：一束光由自然光和线偏振光混合组成。通过旋转的偏振片时，透射光强最大值是最小值的5倍。求入射光中自然光和线偏振光的光强各占总光强的比例。
        - **解**：设入射自然光强 $I_{n0}$，线偏振光强 $I_{p0}$。透射光强 $I = I_{n0}/2 + I_{p0}\cos^2\alpha$。
            $I_{max} = I_{n0}/2 + I_{p0}$，$I_{min} = I_{n0}/2$。
            $I_{max} = 5 I_{min} \implies I_{n0}/2 + I_{p0} = 5(I_{n0}/2) \implies I_{p0} = 2I_{n0}$。
            总光强 $I_{total} = I_{n0} + I_{p0} = 3I_{n0}$。
            自然光占 $I_{n0}/I_{total} = 1/3$。线偏振光占 $I_{p0}/I_{total} = 2/3$。

### 10.2.3. 偏振片的应用
- 光学仪器中的起偏和检偏元件。
- 照相机滤光镜，消除反射眩光。
- 偏光眼镜，观看立体电影。
- 交通应用，减少会车时灯光眩目。

## 10.3. 反射和折射时的偏振

### 10.3.1. 布儒斯特定律 (Brewster's Law)

![](./assets/graph10-5.png)

- **10.3.1.1. 现象描述**
    - 自然光在两种各向同性介质界面反射和折射时，反射光和折射光通常都成为部分偏振光。
    - 反射光中，垂直于入射面的振动分量占优势。
    - 折射光中，平行于入射面的振动分量占优势。
- **10.3.1.2. 起偏角 (布儒斯特角 $i_0$)**
    - 当入射角 $i$ 等于特定的起偏角 $i_0$ 时，反射光成为完全线偏振光，其振动方向垂直于入射面。
    - 布儒斯特定律：$\tan i_0 = \frac{n_2}{n_1}$ ($n_1, n_2$ 分别为入射介质和折射介质的折射率)。
    - 此时，反射光线与折射光线相互垂直，即 $i_0 + \gamma = \pi/2$ ($\gamma$ 为折射角)。
    - 折射光仍为部分偏振光。
- **10.3.1.3. 应用实例**
![](./assets/graph10-6.png)
    - **例2：光线在平行平板内的多次反射折射**
        - **问题**：自然光以布儒斯特角 $i_0$ 从空气射向玻璃平板，在玻璃-空气界面（界面2）的反射光是什么偏振态？
        - **解**：界面1（空气-玻璃）：$\tan i_0 = n_{玻璃}/n_{空气}$，折射角 $\gamma = \pi/2 - i_0$。
            光线以角 $\gamma$ 入射到界面2（玻璃-空气）。界面2的布儒斯特角 $i_0'$ 满足 $\tan i_0' = n_{空气}/n_{玻璃}$。
            因为 $\tan\gamma = \tan(\pi/2 - i_0) = \cot i_0 = n_{空气}/n_{玻璃} = \tan i_0'$。
            所以 $\gamma = i_0'$，即光线以布儒斯特角入射到界面2。
            故界面2的反射光是振动方向垂直于入射面的线偏振光。
- **10.3.1.4. 玻璃片堆 (Pile of Plates)**
    - 利用多次反射和折射，可以使透射光也趋近于线偏振光（振动方向平行于入射面）。
