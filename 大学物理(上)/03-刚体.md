## 3.1. 刚体定轴转动的运动学描述

### 3.1.1. 刚体的定义与基本运动形式
- **刚体 (Rigid Body)**: 在外力作用下，内部任意两点间距离始终保持不变的物体。可视为由无数连续分布的质点组成的特殊质点系。

- **刚体的基本运动形式**:
    1. **平动 (Translation)**: 刚体在运动过程中，其内部任意两点的连线始终保持平行。刚体的平动可简化为质点运动进行处理。
    
       ![](./assets/graph3-2.png)
    
       ![](./assets/graph3-3.png)
    
    2. **定轴转动 (Fixed-Axis Rotation)**: 刚体上所有质点都围绕同一直线（转轴）作圆周运动。
    
       ![](./assets/graph3-4.png)
    
    3. **一般运动**: 可分解为平动与定轴转动的叠加。

### 3.1.2. 描述定轴转动的物理量
- **描述方法**: 刚体定轴转动通常用角位移、角速度和角加速度等角量来描述。
- **定轴转动的特点**:
    - 刚体上所有质点都在与转轴垂直的平面上作圆周运动。
    - 在同一时刻，所有质点的角位移、角速度、角加速度都相同。
- **角速度矢量 ($\vec{\omega}$)**:
    - **方向**: 沿转轴方向，通过右手螺旋法则确定（四指沿转动方向，大拇指指向为 $\vec{\omega}$ 的方向）。
    - **与线速度的关系**: $\vec{v} = \vec{\omega} \times \vec{r}$
- **角加速度矢量 ($\vec{\alpha}$)**:
    - **定义**: $\vec{\alpha} = \frac{d\vec{\omega}}{dt}$
    - **方向**: 在定轴转动中，$\vec{\alpha}$ 的方向沿转轴，与 $\vec{\omega}$ 的方向相同（加速）或相反（减速）。

### 3.1.3. 匀变速定轴转动公式
若刚体绕定轴做匀变速转动（角加速度 $\alpha$ 为常量），其运动公式**与匀变速直线运动类似**：
- $\omega = \omega_0 + \alpha t$
- $\theta = \theta_0 + \omega_0 t + \frac{1}{2}\alpha t^2$
- $\omega^2 = \omega_0^2 + 2\alpha(\theta - \theta_0)$

### 3.1.4. 运动学应用实例
- **问题**: 半径为0.2m的飞轮，初转速为150 r/min，经30s均匀减速至停止。求：(1) 角加速度和转过的圈数；(2) 6s时的角速度；(3) 6s时飞轮边缘上一点的线速度与加速度。
- **解**:
    - (1) 初角速度 $\omega_0 = 5\pi$ rad/s。角加速度 $\alpha = \frac{0-5\pi}{30} = -\frac{\pi}{6}$ rad/s²。转过的总角度 $\theta = \frac{0 - (5\pi)^2}{2\alpha} = 75\pi$ rad，即37.5圈（75π除以一个2π）。
    - (2) 6s时的角速度 $\omega = \omega_0 + \alpha t = 5\pi - \frac{\pi}{6} \times 6 = 4\pi$ rad/s。
    - (3) 6s时边缘点的线速度大小 $v = r\omega = 0.2 \times 4\pi = 2.5$ m/s。切向加速度 $a_t = r\alpha = -0.105$ m/s²。法向加速度 $a_n = r\omega^2 = 31.6$ m/s²。

## 3.2. 刚体定轴转动动力学

### 3.2.1. 力矩 (Moment of Force or Torque)
- **定义**: 力 $\vec{F}$ 对某一点（或某一转轴）的矩，是描述力使物体产生转动效应的物理量。
- **计算**: $\vec{M} = \vec{r} \times \vec{F}$，其中 $\vec{r}$ 是从转轴到力的作用点的矢量。
- **大小**: $M = rF \sin\theta = F \cdot d$，其中 $d$ 为力臂。
- **方向**: 垂直于 $\vec{r}$ 和 $\vec{F}$ 构成的平面，即沿转轴方向。
- **注意**: 只有垂直于转轴的力的分量 $\vec{F}_{\perp}$ 才能产生对该轴的力矩。

### 3.2.2. 转动惯量 (Moment of Inertia)

![](./assets/graph3-5.png)

#### 3.2.2.1. 定义与性质
- **定义**: 刚体绕定轴转动的惯性大小的量度，用 $J$ 表示。
    - **分立质点系**: $J = \sum_i m_i r_i^2$
    - **连续质量分布刚体**: $J = \int r^2 dm$
- **性质**:
    - 与刚体的总质量有关。
    - 与刚体的质量分布有关。
    - 与转轴的位置有关。

#### 3.2.2.2. 常见刚体的转动惯量
| 形状 | 转轴位置 | 转动惯量 (J) |
| :--- | :--- | :--- |
| 细棒 (长l) | 通过中心与棒垂直 | $\frac{1}{12}ml^2$ |
| 细棒 (长l) | 通过端点与棒垂直 | $\frac{1}{3}ml^2$ |
| 薄圆环 (半径r) | 通过中心与环面垂直 | $mr^2$ |
| 薄圆环 (半径r) | 沿直径 | $\frac{1}{2}mr^2$ |
| 薄圆盘 (半径r) | 通过中心与盘面垂直 | $\frac{1}{2}mr^2$ |
| 空心圆筒 (内r₁,外r₂) | 沿几何轴 | $\frac{1}{2}m(r_1^2+r_2^2)$ |
| 实心圆柱 (半径r) | 沿几何轴 | $\frac{1}{2}mr^2$ |
| 球体 (半径r) | 沿直径 | $\frac{2}{5}mr^2$ |
| 球壳 (半径r) | 沿直径 | $\frac{2}{3}mr^2$ |

#### 3.2.2.3. 转动惯量的计算实例

![](./assets/graph3-11.png)

**例1 (分立质点)**: 由长为l的轻杆连接的质点系如图所示，求质点系对过A点且垂直于平面的轴的转动惯量。

    ![](./assets/graph3-12.png)

    - **解**: 根据定义式 $J = \sum_i m_i r_i^2$
    - $J = m_A r_A^2 + m_B r_B^2 + m_C r_C^2 = (2m)l^2 + (3m)(2l)^2 + (4m+5m)(\sqrt{l^2+l^2})^2$
    - $J = 2ml^2 + 12ml^2 + 9m(2l^2) = 32ml^2$

- **例2 (连续体-细棒)**: 计算长为L、质量为m的均质细棒的转动惯量。(1) 对通过杆的一端并与棒垂直的轴；(2) 对通过杆的中心并与棒垂直的轴。

    - **解**: 设棒的线密度为 $\lambda = m/L$。在距离轴x处取一质量元 $dm = \lambda dx = \frac{m}{L}dx$。

    - (1) **轴在端点**:

        ![](./assets/graph3-13.png)

        - $J = \int r^2 dm = \int_0^L x^2 (\frac{m}{L}dx) = \frac{m}{L} [\frac{1}{3}x^3]_0^L = \frac{1}{3}mL^2$

    - (2) **轴在中心**:
      
        ![](./assets/graph3-14.png)
        
        - $J = \int r^2 dm = \int_{-L/2}^{L/2} x^2 (\frac{m}{L}dx) = \frac{m}{L} [\frac{1}{3}x^3]_{-L/2}^{L/2} = \frac{1}{12}mL^2$

### 3.2.3. 定轴转动定律
- **内容**: 刚体绕定轴转动时，其角加速度与所受的合外力矩成正比，与刚体的转动惯量成反比。
- **公式**: $M = J\alpha = J\frac{d\omega}{dt}$
- **类比**:
| 平动 | 转动 |
| :---: | :---: |
| 力 $\vec{F}$ | 力矩 $\vec{M}$ |
| 质量 $m$ | 转动惯量 $J$ |
| 加速度 $\vec{a}$ | 角加速度 $\vec{\alpha}$ |
| $\vec{F} = m\vec{a}$ | $M = J\alpha$ |

### 3.2.4. 转动定律的应用实例
- **问题1**: 转动惯量为 $J$ 的飞轮，初角速度为 $\omega_0$，受到的阻力矩大小与角速度的平方成正比 ($M = -k\omega^2$)。求：(1) 角速度为 $\omega_0/3$ 时的角加速度；(2) 角速度从 $\omega_0$ 减为 $\omega_0/3$ 所需的时间。

- **解**:
    - (1) 由转动定律 $M=J\alpha$，得 $\alpha = M/J = -k\omega^2/J$。代入 $\omega=\omega_0/3$，得 $\alpha = -k\omega_0^2/(9J)$。
    - (2) 由 $-k\omega^2 = J\frac{d\omega}{dt}$，分离变量并积分 $\int_{\omega_0}^{\omega_0/3} \frac{d\omega}{\omega^2} = -\int_0^t \frac{k}{J}dt$，解得 $t = \frac{2J}{k\omega_0}$。
    
- **问题2**: 质量为 $m_1, m_2$ 的物体通过绳子挂在半径分别为 $R_1, R_2$ 的鼓轮上，鼓轮转动惯量为 $J$。求两物体的加速度。

    ![](./assets/graph3-18.png)

- **解**: 分别对两物体和鼓轮列方程：
    - $m_1g - T_1 = m_1a_1$
    - $T_2 - m_2g = m_2a_2$
    - $T_1R_1 - T_2R_2 = J\alpha$
    - 运动学关系: $a_1 = R_1\alpha, a_2 = R_2\alpha$
    - 联立解得: $a_1 = \frac{m_1R_1 - m_2R_2}{m_1R_1^2 + m_2R_2^2 + J}gR_1$，$a_2 = \frac{m_1R_1 - m_2R_2}{m_1R_1^2 + m_2R_2^2 + J}gR_2$。

## 3.3. 刚体转动中的功和能

### 3.3.1. 力矩的功
- **定义**: 刚体在合外力矩 $M$ 作用下转过角位移 $\theta$ 时，合外力矩所做的功。
- **元功**: $dW = Md\theta$
- **总功**: $W = \int_{\theta_1}^{\theta_2} Md\theta$

### 3.3.2. 转动动能与动能定理
- **转动动能**: 刚体绕定轴转动时具有的动能。$E_k = \sum \frac{1}{2}\Delta m_i v_i^2 = \frac{1}{2}(\sum \Delta m_i r_i^2)\omega^2 = \frac{1}{2}J\omega^2$。
- **刚体定轴转动动能定理**: 合外力矩对刚体所做的功，等于刚体转动动能的增量。
    - $W = \frac{1}{2}J\omega_2^2 - \frac{1}{2}J\omega_1^2 = \Delta E_k$

### 3.3.3. 刚体的重力势能与机械能
- **刚体的重力势能**: 等于将刚体质量全部集中在质心处的质点的重力势能。
    - $E_p = mgh_c$，其中 $h_c$ 为质心的高度。
- **刚体的机械能**: $E = E_{k,转动} + E_p = \frac{1}{2}J\omega^2 + mgh_c$

### 3.3.4. 功能原理与机械能守恒
- **功能原理**: 外力矩（不含重力）对刚体做的功等于系统机械能的增量。
    - $\int_{\theta_1}^{\theta_2} M_{外}d\theta = E_2 - E_1 = (E_{k2}+E_{p2}) - (E_{k1}+E_{p1})$
- **机械能守恒**: 若只有重力（或其他保守力）的力矩做功，则系统机械能守恒。

### 3.3.5. 功和能的应用实例
- **问题1**: 质量为M、半径为R的圆盘，盘上绕有细绳，一端挂有质量为m的物体。求物体由静止下落h高度时的速率。

    ![](./assets/graph3-19.png)

- **解 (系统机械能守恒)**: 系统初态机械能为0。末态物块动能 $\frac{1}{2}mv^2$，圆盘转动动能 $\frac{1}{2}J\omega^2 = \frac{1}{2}(\frac{1}{2}MR^2)(\frac{v}{R})^2 = \frac{1}{4}Mv^2$，物块重力势能减少 $mgh$。
  
    - $0 = (\frac{1}{2}mv^2 + \frac{1}{4}Mv^2) - mgh$
    - 解得 $v = \sqrt{\frac{mgh}{m/2+M/4}}$ (注：原文答案有误，这是正确结果)。
    
- **问题2**: 长为2l、质量为3m的细杆，两端固定质量为2m和m的小球，可绕中心o点转动。求从水平位置静止释放，转到垂直位置时的角速度。

    ![](./assets/graph3-20.png)

![](./assets/graph3-21.png)
  
![](./assets/graph3-22.png)
  
- **解 (系统机械能守恒)**: 取最低点为零势能面。
  
    ![](./assets/graph3-23.png)
    
    - 初态势能: $E_{p1} = 2mgl + mgl = 3mgl$ (杆的质心在o点，势能为0)。
    - 末态势能: $E_{p2} = mg(2l) = 2mgl$。
    - 末态动能: $E_{k2} = \frac{1}{2}J_o\omega^2$。转动惯量 $J_o = 2m l^2 + m l^2 + \frac{1}{12}(3m)(2l)^2 = 4ml^2$。
    - 由 $E_{p1} = E_{p2} + E_{k2}$，得 $3mgl = 2mgl + \frac{1}{2}(4ml^2)\omega^2$。
    - 解得 $\omega = \sqrt{\frac{g}{2l}}$。

## 3.4. 角动量与角动量守恒

### 3.4.1. 角动量 (Angular Momentum)
- **质点的角动量**: 质点的动量 $\vec{p}$ 对某一点O的矩，也称动量矩。
    - $\vec{L} = \vec{r} \times \vec{p} = \vec{r} \times m\vec{v}$
- **刚体对定轴的角动量**: 刚体绕定轴转动时，其总角动量为所有质元角动量之和。
    - $L_z = (\sum \Delta m_i r_i^2)\omega = J_z\omega$
    - 其方向沿转轴方向。

### 3.4.2. 角动量定理
- **微分形式**: 作用于刚体的合外力矩，等于刚体角动量对时间的变化率。
    - $M = \frac{d(J\omega)}{dt} = \frac{dL}{dt}$
- **积分形式**: 刚体所受合外力矩的冲量矩，等于刚体角动量的增量。
    - $\int_{t_1}^{t_2} Mdt = J\omega_2 - J\omega_1 = \Delta L$

### 3.4.3. 角动量守恒定律
- **内容**: 如果作用于一个系统的合外力矩为零，那么该系统的总角动量保持不变。
- **公式**: 若 $M_{合外}=0$，则 $L = J\omega = \text{常量}$。
- **应用**: 若系统在作用过程中转动惯量发生变化（如人收缩手臂），则角速度会相应改变，以保证角动量守恒 ($J_1\omega_1 = J_2\omega_2$)。

### 3.4.4. 角动量定律的应用实例
- **问题1**: 质量m、半径R的均质圆盘，初角速度 $\omega_0$，与桌面摩擦系数为μ。求圆盘停止转动所需的时间。

    ![](./assets/graph3-24.png)

- **解**:
    - 摩擦力矩: $M_f = \int_0^R -r(\mu g dm) = -\frac{2}{3}\mu mgR$ (为恒定力矩)。
    - 应用角动量定理: $\int_0^t M_f dt = L_f - L_i \Rightarrow M_f \cdot t = 0 - J\omega_0$
    - $(-\frac{2}{3}\mu mgR) \cdot t = -(\frac{1}{2}mR^2)\omega_0$
    - 解得 $t = \frac{3R\omega_0}{4\mu g}$。
    
- **问题2**: 质量为 $m_2$ 的小滑块以速度 $v_1$ 垂直碰撞静止细棒（质量 $m_1$，长 $l$）的末端，碰后滑块以速度 $v_2$ 反弹。求棒从开始转动到因摩擦停止所需的时间。

    ![](./assets/graph3-25.png)

- **解**:
  
    - **第一过程 (碰撞)**: 时间极短，忽略摩擦力矩，系统角动量守恒。
        - $m_2v_1l = -m_2v_2l + J\omega = -m_2v_2l + (\frac{1}{3}m_1l^2)\omega$。可解出碰后瞬间棒的角速度 $\omega$。
    - **第二过程 (减速)**: 棒受到恒定的摩擦力矩 $M_f = -\frac{1}{2}\mu m_1 gl$。
        - 应用角动量定理: $M_f \cdot t = 0 - J\omega$
        - $(-\frac{1}{2}\mu m_1 gl)t = -(\frac{1}{3}m_1l^2)\omega$。
    - 联立两式消去 $\omega$，解得 $t=2m_2\frac{v_1+v_2}{\mu m_1 g}$。
    
- **问题3**: 质量为m、速度为v的子弹射入悬挂杆（质量M，长l）内（射入点距支点a），使杆摆起30°。求子弹初速度v。

    ![](./assets/graph3-26.png)

- **解**:
    - **第一过程 (碰撞)**: 子弹与杆系统角动量守恒。
        - $mva = J_{总}\omega = (\frac{1}{3}Ml^2+ma^2)\omega$
    - **第二过程 (上摆)**: 子弹与杆系统机械能守恒。
        - $\frac{1}{2}J_{总}\omega^2 = (m+M)g\Delta h_c = mga(1-\cos 30^\circ) + Mg\frac{l}{2}(1-\cos 30^\circ)$
    - 联立两式即可解出 $v$。

### 3.4.5. 综合讨论题
- **问题1**: 人站在转台上，双臂伸直水平举哑铃。当他把哑铃收缩到胸前时，人、哑铃和平台组成的系统：
- **答案 (C)**: **机械能不守恒，角动量守恒**。因为人做功改变了系统的动能（内力做功），但系统所受合外力矩为零，故角动量守恒。
- **问题2**: 人造地球卫星绕地球作椭圆轨道运动，则卫星的：
- **答案 (C)**: **角动量守恒，动能不守恒**。因为卫星只受地球的万有引力，该力始终指向地心，故对地心的力矩为零，角动量守恒。但引力会做功，导致卫星在近地点和远地点的速度不同，动能不守恒。
