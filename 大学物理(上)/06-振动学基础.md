# 6. 振动学基础

## 6.1. 简谐振动 (SHM) 的描述

![](./assets/graph6-1.png)

### 6.1.1. 简谐振动的定义与运动学方程
- **振动**: 一个物理量随时间做周期性或准周期性变化。
- **简谐振动(SHM)**: 物体的位移随时间按余弦（或正弦）规律变化的运动，是最简单、最基本的振动。
- **动力学特征**: 物体所受的回复力与偏离平衡位置的位移成正比，方向相反。$F = -kx$。
- **动力学方程**: $m\frac{d^2x}{dt^2} = -kx \Rightarrow \frac{d^2x}{dt^2} + \omega^2 x = 0$，其中 $\omega^2 = k/m$。
- **运动学方程 (解)**:
    - **位移**: $x = A\cos(\omega t + \varphi)$
    - **速度**: $v = \frac{dx}{dt} = -\omega A\sin(\omega t + \varphi)$
    - **加速度**: $a = \frac{dv}{dt} = -\omega^2 A\cos(\omega t + \varphi) = -\omega^2 x$

![](./assets/graph6-2.png)

### 6.1.2. 简谐振动的特征量

1.  **振幅 (Amplitude, A)**: 振动物体离开平衡位置的最大位移的绝对值。
2.  **角频率 (Angular Frequency, ω)**: 描述振动快慢的物理量。
    - **周期 (Period, T)**: $T = \frac{2\pi}{\omega}$。
    - **频率 (Frequency, ν)**: $\nu = \frac{1}{T} = \frac{\omega}{2\pi}$。
    - 对于弹簧振子，固有角频率 $\omega = \sqrt{k/m}$。
3.  **相位 (Phase)**:
    - **相位**: $(\omega t + \varphi)$，决定了振动物体在t时刻的运动状态(x, v, a)。
    - **初相 (Initial Phase, φ)**: t=0时刻的相位。

### 6.1.3. 振幅A和初相φ的确定
A和φ是由振动的初始条件（$t=0$时的位置$x_0$和速度$v_0$）决定的。
- $x_0 = A\cos\varphi$
- $v_0 = -\omega A\sin\varphi$
- 联立解得: $A = \sqrt{x_0^2 + (v_0/\omega)^2}$，$\tan\varphi = -v_0/(\omega x_0)$。

### 6.1.4. 旋转矢量表示法
- **方法**: 用一个以角速度ω逆时针匀速旋转的矢量 $\vec{A}$ 来表示简谐振动。
    - 矢量的**模**等于振动的**振幅A**。
    - 矢量旋转的**角速度**等于振动的**角频率ω**。
    - t=0时矢量与x轴的**夹角**等于振动的**初相φ**。
- **投影**: 旋转矢量在x轴上的投影点的运动规律就是 $x = A\cos(\omega t + \varphi)$。
- **优点**: 直观，便于求解初相和进行振动合成。

    ![](./assets/graph6-5.png)
    
    - **解**: (1) $\omega=\pi$ rad/s。由 $x_0=A\cos\varphi$ 和 $v_0>0$ 得 $\varphi=-\pi/3$。表达式为 $x = 0.12\cos(\pi t - \pi/3)$ m。
    - (2) t=0.5s时，$x=0.104$m, $v=-0.189$m/s, $a=-1.03$m/s² (原文a计算有误)。
    - (3) x=-6cm, v<0 对应相位 $2\pi/3$。第一次回到平衡位置(v<0)对应相位 $\pi/2$。这是不可能的，应是回到v>0的平衡位置，对应相位 $3\pi/2$。旋转矢量需转过 $\Delta\theta = 3\pi/2 - 2\pi/3 = 5\pi/6$。所需时间 $\Delta t = \Delta\theta/\omega = (5\pi/6)/\pi = 5/6 \approx 0.83$ s。
    
- **例2**: 两同频同幅振动，当振子1在 $x_1=A/2$, v<0处时，振子2在 $x_2=0$, v>0处。求相位差。

## 6.2. 简谐振动的判据与实例

### 6.2.1. 简谐振动的判据
1.  **运动学判据**: 运动方程为 $x=A\cos(\omega t+\varphi)$。
2.  **动力学判据**: 回复力满足 $F=-kx$。
3.  **微分方程判据**: 运动的微分方程为 $\frac{d^2x}{dt^2}+\omega^2x=0$。

### 6.2.2. 实例分析
- **单摆**: 在小角度摆动时($\sin\theta \approx \theta$)，回复力矩 $-mg\sin\theta \cdot l \approx -(mgl)\theta$。动力学方程为 $J\alpha = -(mgl)\theta \Rightarrow ml^2\frac{d^2\theta}{dt^2} = -(mgl)\theta \Rightarrow \frac{d^2\theta}{dt^2} + \frac{g}{l}\theta = 0$。此为SHM，周期 $T=2\pi\sqrt{l/g}$。
- **比重计**: 设比重计在平衡位置浸入体积为V，在位移x处，所受合力为 $F_{合} = F_{浮}-G = \rho g(V+\pi(d/2)^2x) - mg$。因平衡时 $mg=\rho gV$，故 $F_{合} = -(\rho g \pi d^2/4)x$。此为SHM，周期 $T=2\pi\sqrt{m/k} = \frac{4}{d}\sqrt{\frac{\pi m}{\rho g}}$。

## 6.3. 简谐振动的能量

![](./assets/graph6-10.png)

### 6.3.1. 动能、势能与总能量
- **动能**: $E_k = \frac{1}{2}mv^2 = \frac{1}{2}m\omega^2A^2\sin^2(\omega t+\varphi)$
- **势能**: $E_p = \frac{1}{2}kx^2 = \frac{1}{2}kA^2\cos^2(\omega t+\varphi)$
- **总机械能**: $E = E_k + E_p = \frac{1}{2}kA^2\sin^2(\omega t+\varphi) + \frac{1}{2}kA^2\cos^2(\omega t+\varphi) = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2A^2$。
- **结论**: 在没有阻尼的情况下，简谐振动的总机械能守恒。动能和势能相互转化，但其变化频率是振动频率的2倍。 

### 6.3.2. 应用实例
- **例6-5**: m=0.1kg, A=0.01m, $a_{max}=4.0$ m/s²。求(1)周期T; (2)最大动能; (3)总能量; (4)动能=势能时的位置。
    - **解**: (1) $a_{max}=A\omega^2 \Rightarrow \omega=\sqrt{4.0/0.01}=20$ rad/s。$T=2\pi/\omega \approx 0.314$s。
    - (2) 最大动能 $E_{k,max} = \frac{1}{2}mv_{max}^2 = \frac{1}{2}m(A\omega)^2 = \frac{1}{2}(0.1)(0.01\times 20)^2 = 0.002$ J。
    - (3) 总能量 $E = E_{k,max} = 0.002$ J。
    - (4) 当 $E_k=E_p$ 时，$E_p=E/2=0.001$ J。由 $E_p=\frac{1}{2}kx^2=\frac{1}{2}m\omega^2x^2$，得 $x = \sqrt{2E_p/(m\omega^2)} = \sqrt{2(0.001)/(0.1\cdot 20^2)} \approx \pm 0.00707$ m。

## 6.4. 振动的合成

### 6.4.1. 同方向同频率简谐振动的合成
- **原理**: 两个同方向、同频率的简谐振动 $x_1 = A_1\cos(\omega t + \varphi_1)$ 和 $x_2 = A_2\cos(\omega t + \varphi_2)$ 合成后，仍为同频率的简谐振动 $x = A\cos(\omega t + \varphi)$。

    ![](./assets/graph6-11.png)

- **合成振幅与初相**:
    - $A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos(\varphi_2 - \varphi_1)}$
    - $\tan\varphi = \frac{A_1\sin\varphi_1 + A_2\sin\varphi_2}{A_1\cos\varphi_1 + A_2\cos\varphi_2}$
    
- **特殊情况**:
    - **同相** ($\Delta\varphi=2k\pi$): $A = A_1+A_2$ (振动加强)。
    - **反相** ($\Delta\varphi=(2k+1)\pi$): $A = |A_1-A_2|$ (振动减弱)。

### 6.4.2. 相互垂直简谐振动的合成
- **一般情况**: 合成轨迹为椭圆，方程为 $\frac{x^2}{A_1^2} + \frac{y^2}{A_2^2} - \frac{2xy}{A_1A_2}\cos(\Delta\varphi) = \sin^2(\Delta\varphi)$。

- **特殊情况 (频率相同)**:
  
    ![](./assets/graph6-12.png)
    
    - $\Delta\varphi=0$ 或 $\pi$: 合成轨迹为直线。
    - $\Delta\varphi=\pi/2$ 或 $3\pi/2$: 合成轨迹为标准椭圆。
    
- **频率成简单整数比**: 合成轨迹为稳定的、封闭的复杂曲线，称为李萨如图形。

### 6.4.3. 应用实例
- **例题1**: 两个同方向的简谐振动曲线如图所示(反相)，求合振动表达式。
    - **解**: 由图可知，两振动反相，频率相同。合振幅 $A=|A_2-A_1|$。合振动的初相与振幅较大的分振动相同。若$A_2>A_1$，则合振动初相为$\varphi_2 = -\pi/2$。合振动表达式为 $x = |A_2-A_1|\cos(\frac{2\pi}{T}t - \frac{\pi}{2})$。
- **例题2**: 合振动A=20cm，与分振动1的相位差为$\pi/6$。分振动1的振幅$A_1=10\sqrt{3}$cm。求$A_2$和两分振动的相位差。
    - **解**: 利用矢量三角形中的余弦定理：$A_2^2 = A^2+A_1^2 - 2AA_1\cos(\pi/6) = 20^2 + (10\sqrt{3})^2 - 2(20)(10\sqrt{3})(\sqrt{3}/2) = 100$。所以 $A_2=10$ cm。
    - 再用正弦定理：$\frac{A_2}{\sin(\pi/6)} = \frac{A}{\sin(\pi-\Delta\varphi)}$。可求出两分振动的相位差 $\Delta\varphi$。
