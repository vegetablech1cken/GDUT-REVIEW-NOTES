# 2.1 物理层的基本概念

1. **核心功能**
    
    - 信号转换：实现数据链路层 “帧” 与物理介质可传输信号（电 / 光 / 无线电）的双向转换
        
    - 介质管理：定义物理介质的使用规则（时序、速率同步等）
        
    - 接口规范：规定设备与介质的连接标准（如 RJ45、SFP 光口）
        
2. **关键特性参数**
    
    - 机械特性：接口物理结构（如 RJ45 的 8 根引脚）
        
    - 电气特性：信号电压范围、传输速率等（如以太网高电平 + 2.5V~+5V）
        
    - 功能特性：接口引脚用途（如 RJ45 的 1/2 引脚发送数据，3/6 引脚接收数据）
        
    - 规程特性：信号传输流程（如串口通信的 “握手” 步骤）
        
3. **物理层的主要任务**
    
    - 机械特性 ：指明接口所用接线器的形状和尺寸、引线数目和排列、固定和锁定装置等。
        
    - 电气特性：指明在接口电缆的各条线上出现的电压的范围。
        
    - 功能特性：指明某条线上出现的某一电平的电压表示何种意义。
        
    - 过程特性 ：指明对于不同功能的各种可能事件的出现顺序。
        
    - 传输方式转换：指明数据在计算机内部的并行传输与在传输媒体上的串行传输的转换方式。
        
4. **常用物理层相关标准：RJ45**
    
    - 机械特性：8P8C 水晶头，568A/568B 线序
        
    - 电气特性：额定电流 1.5A，额定电压 125VAC，绝缘电阻 ≥500MΩ，接触电阻 ≤20MΩ
        
    - 功能特性：差分信号，电势差 <1.6V 表示 “0”，>8V 表示 “1”

6. **物理层设备**
    
    - 中继器：放大衰减信号，延长传输距离（仅用于双绞线 / 同轴电缆）
        
    - 集线器（Hub）：多端口中继器，共享带宽，易冲突，已被交换机取代
        
7. **数据传输基本概念**
    
    - 基带传输：直接传输数字信号（如以太网），需编码（如曼彻斯特编码）
        
    - 宽带传输：数字信号调制到模拟载波传输（如 WiFi），需调制解调
        
    - 通信方式：单工（广播）、半双工（对讲机）、全双工（以太网）
        

# 2.2 数据通信的基础知识

1. **数据通信系统模型**
    
    - 组成：源系统（发送端）、传输系统（网络）、目的系统（接收端）
        
    - 核心术语：
        
        - 消息：话音、文字、图像、视频等原始信息
            
        - 数据：运送消息的实体，用特定方式表示的消息
            
        - 信号：数据的电气或电磁表现
            
            - 模拟信号：参数连续变化
                
            - 数字信号：参数离散变化
                
        - 码元：代表不同离散数值的基本波形
            
        - 信源：产生和发送数据的源头
            
        - 信宿：接收数据的终点
            
        - 信道：信号传输的通道
            
2. **信道基本概念**
    
    - 单向通信（单工）：只能单向传输，如电视广播
        
    - 双向交替通信（半双工）：双方可发送但不能同时，如对讲机
        
    - 双向同时通信（全双工）：双方可同时收发，如电话
        
3. **调制与编码**
    
    - 基带调制：对数字信号波形进行变换，仍为基带信号，也称编码
        
    - 带通调制：用载波将基带信号频谱搬移到高频，转为模拟信号
        
    - 常用编码方式：
        
        表格
        
        复制
        
        |编码方式|特点|
        |:--|:--|
        |不归零制|正电平 1，负电平 0，无自同步能力|
        |归零制|正脉冲 1，负脉冲 0|
        |曼彻斯特编码|位中心跳变，上跳 0 下跳 1，自带时钟，10BASE-T 以太网用|
        |差分曼彻斯特|位中心必跳变，位开始跳变代表 0，不跳变代表 1，Token Ring 用|
        
    - 带通调制：调幅（AM）、调频（FM）、调相（PM）
        
    - 正交振幅调制（QAM）：同时调幅调相，16QAM 每符号 4bit，速率提高 4 倍
        
4. **信道极限容量**
    
    - 信噪比：S/N，单位 dB，信噪比(dB)=10log10​(S/N)
        
    - 香农定理：极限传输速率C=Wlog2​(1+S/N)
        
        - W：带宽（Hz）
            
        - S：信号功率
            
        - N：噪声功率
            
    - 结论：带宽或信噪比越大，极限速率越高；实际速率低于香农极限
        

# 2.3 物理层下面的传输媒体

1. **导引型传输媒体**
    
    - 双绞线：
        
        - UTP：无屏蔽，成本低，家用、办公
            
        - STP：金属屏蔽，抗干扰，机房、工业
            
        - 标准：EIA/TIA-568，常用 CAT5（100MHz）、CAT5E（125MHz）、CAT6（250MHz）
            
    - 同轴电缆：
        
        - 50Ω：数字传输，LAN
            
        - 75Ω：模拟传输，CATV
            
    - 光纤：
        
        - 单模：芯径 9µm，激光光源，千米级，低色散，高成本
            
        - 多模：芯径 50/62.5µm，LED 光源，百米级，高色散，低成本
            
        - 波段：850nm、1300nm、1550nm 损耗最小，带宽 25~30THz
            
        - 优点：带宽大、损耗低、抗干扰、保密好
            
        - 缺点：接口贵、易断、维护难
            
2. **非导引型传输媒体**
    
    - 短波：靠电离层反射，低速几十~几千 bit/s
        
    - 微波：300MHz~300GHz，直线传播，50km 需中继，多径效应明显
        
    - 卫星通信：同步卫星三颗覆盖全球，用于 GPS、北斗、远程通信
        

# 2.4 信道复用技术

1. **频分复用（FDM）**
    
    - 把总带宽划分为多个子频带，用户独占子频带，并行传输
        
    - 需保护频带，实时性好，用于广播、有线电视
        
2. **同步时分复用（TDM）**
    
    - 时间划分为帧，每帧固定时隙，时隙周期性分配
        
    - 时隙可能空闲，利用率低，用于 E1、T1
        
3. **统计时分复用（STDM）**
    
    - 时隙动态分配，数据贴标签，帧满即发
        
    - 利用率高，用于以太网、Wi-Fi
        
4. **波分复用（WDM）**
    
    - 光域 FDM，不同波长光信号同纤传输
        
    - CWDM（20nm）、DWDM（0.4/0.8nm），单纤 40Tbps
        
5. **码分复用（CDM/CDMA）**
    
    - 每个用户分配唯一 m bit 码片序列，1 发原序列，0 发反码
        
    - 码片序列需正交，规格化内积为 0
        
    - 接收端用对应码片与叠加信号求内积解扩
        
    - 特点：抗干扰、保密强、需功率控制，用于 3G、GPS
        
# 2.5 数字传输系统

1. **系统定位**
    
    - 流程：信源编码→信道编码→调制→信道→解调→信道译码→信源译码
        
2. **数字 vs 模拟**
    
    - 数字：再生中继、可加密、易 TDM、包交换
        
    - 模拟：放大噪声、难纠错、多 FDM、电路交换
        
3. **关键指标**
    
    - 比特率Rb​（bit/s）、波特率Rs​（Baud），Rb​=Rs​log2​M
        
    - 频谱效率η=Rb​/B（bit/s/Hz）
        
4. **码型变换**
    
    - NRZ-L、Manchester、HDB3、8B/10B
        
5. **同步技术**
    
    - 位同步：PLL、CDR
        
    - 帧同步：FAW
        
    - 网同步：1588v2 PTP
        
6. **差错控制**
    
    - ARQ、FEC、HARQ
        

# 2.6 宽带接入技术

1. **概念**
    
    - 宽带：下行≥25Mbit/s，上行≥3Mbit/s（ITU）；我国下行≥100Mbit/s 称基础宽带，≥1000Mbit/s 称千兆宽带
        
2. **技术分类**
    
    表格
    
    复制
    
    |介质|技术|关键参数|
    |:--|:--|:--|
    |铜线|xDSL|ADSL2+（下行 24Mbit/s）、VDSL2（下行 100Mbit/s@300m）、G.fast（1Gbit/s@100m）|
    |光纤|FTTx|GPON（下行 2.5Gbit/s）、XG-PON（下行 10Gbit/s）、XGS-PON（对称 10Gbit/s）|
    |同轴|DOCSIS|3.1（下行 10Gbit/s）、4.0（对称 10Gbit/s）|
    |无线|FBWA|5G NR FWA（n78 频段 1.6Gbit/s@1km）、Wi-Fi 6（理论 9.6Gbit/s）|
    
3. **ADSL 技术**
    
    - 非对称：下行>>上行
        
    - 频分：0–4kHz 电话，>40kHz 数据
        
    - DMT：把 1.1MHz 划分为 256 子信道，4kHz/子信道，25 上行 249 下行
        
4. **HFC 网**
    
    - 光纤到节点，同轴到户
        
    - 频分：5–65MHz 上行，87–1000MHz 下行
        
    - 设备：Cable Modem，下行 3–30Mbit/s，上行 0.2–2Mbit/s
        
5. **FTTx 技术**
    
    - FTTH：光纤到户
        
    - FTTB：光纤到大楼
        
    - FTTC：光纤到路边
        
    - PON：无源光网络，ODN 分光，一户一纤
        
6. **演进趋势**
    
    - 光纤化：FTTH 渗透率提升，XGS-PON 标配
        
    - 云化：vOLT、vBNG
        
    - 协同化：FTTR 全屋千兆
        
    - 高速化：50G PON 2025 商用，对称 50Gbit/s
