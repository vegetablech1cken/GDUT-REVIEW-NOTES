## 📖 使用说明

### 对于已完成实验

1. **克隆仓库到本地**

   ```bash
   git clone https://github.com/BI7KHI/GDUT_University-Physics-Experiment-Report.git
   ```
2. **查看实验数据**

   - Excel文件包含完整的数据处理和图表
   - 图片文件展示实验结果和过程
   - 部分实验包含Markdown格式的实验报告
3. **使用实验模板**

   - 下载对应的Excel文件
   - 修改数据为你自己的实验数据
   - Excel会自动重新计算和更新图表

### 对于未完成实验

欢迎贡献！你可以：

- 补充缺失的实验数据
- 添加新的实验报告
- 改进现有的数据处理方法

## 贡献指南

### 如何贡献实验数据

1. Fork 这个仓库
2. 创建你的功能分支 (`git checkout -b experiment/实验名称`)
3. 提交你的更改 (`git commit -m 'Add 实验名称'`)
4. 推送到分支 (`git push origin experiment/实验名称`)
5. 创建一个 Pull Request

### 文件命名规范

- **Excel文件**: `实验编号 实验名称.xlsx`
- **图片文件**: `实验编号 实验名称.jpg/png`
- **Markdown报告**: `实验编号 实验名称.md`
- **文件夹**: `实验编号 实验名称/`
